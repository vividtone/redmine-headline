Redmineの画面に任意のCSSとJavaScriptを注入してカスタマイズが行えるView Customizeプラグインの v2.7.0 リリース。

新機能は ①カスタマイズ適用対象プロジェクトを限定できる Project Pattern の追加 ②カスタマイズが全ページ対象の場合に Path Pattern省略可 — の2つ。
